# Swapi React

A simple UI to explore the Star Wars API.

## Getting started

**Requirements**

- Node 20+

### Run in dev mode

```
# only the first time
npm i

npm run dev
```

### Build project

```
# only the first time
npm i

npm run build
```

To test locally, in addition to preview mode of vite, serve package can be used as follow.

```
npx serve build/client
```
